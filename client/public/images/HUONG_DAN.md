## 🌐 Nguồn tải ảnh

1. **Cellphones.com.vn** - Bạn có thể lấy ảnh từ trang web này
   - Chuột phải → Save Image As
   - Đặt tên theo danh sách trên

2. **Unsplash.com** - Ảnh miễn phí chất lượng cao
   - Tìm "iPhone", "Samsung" v.v.

3. **Pexels.com** - Ảnh miễn phí

4. **Pixabay.com** - Ảnh miễn phí

## 📝 Cách thêm ảnh

1. Tải ảnh từ các nguồn trên
2. Đổi tên ảnh thành tên tệp tương ứng (từ danh sách)
3. Copy vào thư mục: `client\public\images\products\`
4. Chạy lại dự án React

## ✅ Kiểm tra

Nếu ảnh không hiển thị, có thể:
- Kiểm tra tên tệp có đúng không (phân biệt chữ hoa/thường)
- Kiểm tra tệp có nằm trong đúng thư mục không
- Kiểm tra server React có chạy không (npm start)

## 💡 Mẹo

- Kích thước ảnh tốt: 400x400px hoặc 800x600px
- Định dạng: JPG, PNG
- File size: < 500KB mỗi ảnh để load nhanh

---

**Lưu ý:** Các URL CDN sẽ được thay thế bằng đường dẫn local khi bạn thêm ảnh vào thư mục này.
